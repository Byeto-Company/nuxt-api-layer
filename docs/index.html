composables/useCreate

Composable for creating API resources with Axios + TanStack Query mutation.

composables/useCreate
Parameters
options (Object) Options for configuring the API request.
Name Description
options.customResource Object? Optional custom resource config.
options.customResource.name string? Optional resource name (used as mutation key).
options.customResource.path string The API endpoint path for the resource.
options.urlSearchParams ComputedRef<Record<any, any>>? Optional search params passed as query.
options.axiosInstance AxiosInstance? Optional Axios instance (defaults to Nuxt global $axios).
options.axiosOptions Omit<AxiosRequestConfig, "params">? Optional Axios request config.
options.mutationOptions Partial<Omit<UseMutationOptions<TResponse>, ("queryKey" | "queryFn")>>? Extra mutation options from TanStack Query.
options.contentType ("json" | "form") (default "json") Content type for the request body.
options.handleError boolean? Whether errors should be handled by a global handler.
options.authorization boolean? Whether to include authorization headers.
Returns
UseMutationResult<TResponse, ApiError, TRequest>: A TanStack Query mutation object.

options.authorization boolean? Whether to include authorization headers.
